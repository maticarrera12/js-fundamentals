# dev-path

Material de estudio para aprender JavaScript y TypeScript desde cero.
Cada archivo combina explicación, ejemplos ejecutables y ejercicios.

---

## Antes de empezar

Instalá Node.js, pnpm y tsx siguiendo las instrucciones en [SETUP.md](./SETUP.md).

---

## Estructura

```
dev-path/
├── javascript/
│   ├── basic/          → sintaxis, tipos, estructuras, funciones, clases
│   └── intermediate/   → async, prototipos, DOM, generadores, regex
└── typescript/
    └── *.ts / *.mts    → type system, generics, tipos avanzados, clases
```

---

## Por dónde empezar

1. **[JavaScript Roadmap](./javascript/ROADMAP.md)** — empezá acá
2. **[TypeScript Roadmap](./typescript/ROADMAP.md)** — después de completar JavaScript

No saltes a TypeScript sin tener sólido el JavaScript. El orden importa.
